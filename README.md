## Datasets

The datasets used in this study and their sources are listed below:

| Dataset | Type | Description |
|--------|------|-------------|
| [Aalto Dataset](https://research.aalto.fi/en/datasets/iot-devices-captures/) | Benign | Contains network traffic captures from IoT devices under normal (benign) operating conditions. |
| [UNSW Dataset](https://iotanalytics.unsw.edu.au/iottraces.html) | Benign | Includes IoT traffic traces collected from real-world environments. |
| [CIC-IoT Dataset](https://www.unb.ca/cic/datasets/iotdataset-2022.html) | Mixed | A comprehensive dataset with multiple IoT attack scenarios and benign traffic. |
| [BoT-IoT Dataset](https://research.unsw.edu.au/projects/bot-iot-dataset) | Mixed | Focuses on botnet-related attacks alongside normal IoT network traffic. |
| [IoT-23 Dataset](https://www.stratosphereips.org/datasets-iot23) | Mixed | Contains labeled IoT traffic with various malware and attack behaviors along with benign data. |
